# Difference-Objects-Classes
Discussion 1: Analyze the Difference Between Objects and Classes with Examples



 Difference-Objects-Classes

This project demonstrates the difference between classes and objects in Java.

- **Car.java** → defines the Car class (blueprint)
- **TestCar.java** → creates two Car objects (car1 and car2) and displays their info





In Java a class is a plan. It shows what something will have and what it can do. In my program I made a class called Car. It says every car has a color and a speed and can show its information. The class is not a real car.

An object is a real thing made from a class. I made two cars. car1 is red and goes 60. car2 is blue and goes 100. They are different even though they come from the same class.

The main difference is a class is the plan and an object is the real thing. You can make many objects from one class. My program shows the class is the blueprint and the objects are the cars we can use.
